# 📖 About The Project

Shiv Shakti App is a modern AI-powered workshop and shop management platform designed for automobile workshops, garages, service centers, and local businesses.

The platform helps shop owners manage:

✅ Customers  
✅ Vehicles  
✅ Workshop Services  
✅ Inventory & Spare Parts  
✅ Billing & Invoices  
✅ Digital Khata Book  
✅ Business Analytics  
✅ AI Insights  

All from a single dashboard.

# 🚀 Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/shiv-shakti-app.git
```

## 2️⃣ Backend Setup

```bash
cd Backend

npm install

npm run dev
```

## 3️⃣ Frontend Setup

```bash
cd Frontend

npm install

npm run dev
```

# 🔑 Environment Variables
